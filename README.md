# TFG: Desigualdad de la renta equivalente en España (2016–2021)

Código desarrollado para el Trabajo de Fin de Grado.

Contenido:
- Carga y limpieza de datos
- Índices de desigualdad 
- Análisis descriptivo
- Modelos de regresión logística
- Curvas ROC y capacidad predictiva

Nota:
Para ejecutar el script es necesario modificar la variable `ruta_base`
indicando la ubicación local de los microdatos.
